Ship of Tools HTML preview fixture

This file exists to exercise the o-key handoff that opens an text/html preview in the system browser. The chrome's markdown pipeline doesn't render CSS or fetch remote scripts, so a real browser is the place to see this rendered properly.

Why the browser, not the chrome?

Three things on this page rely on the browser:

  • CSS layout (the centred 720px column, table borders, code styling).
  • MathJax: $e^{i\pi} + 1 = 0$ and the display block below.
  • Network: the MathJax script loads from a CDN.

Display math via MathJax:

$$ \int_0^1 x^2 \, dx = \frac{1}{3} \qquad \sum_{k=1}^{n} k = \frac{n(n+1)}{2} $$

Code sample

fn open_html_in_browser(html_bytes: &[u8]) -> std::io::Result<()> {
    let path = std::env::temp_dir()
        .join(format!("sot-preview-{}.html", unique_suffix()));
    std::fs::write(&path, html_bytes)?;
    spawn_default_handler(&path)
}
