# Intent

Small great-circle navigation toolkit: named waypoints, pairwise distance and
bearing, and polyline route length. Spherical-Earth accuracy is accepted
(≈0.5 %); no ellipsoid corrections, no I/O — this stays a pure-math core that
larger routing tools can build on.
